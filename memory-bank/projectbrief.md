# Project Brief: provision-redhat-app

## Overview
Develop an Ansible role to deploy Red Hat Ansible Automation Platform (AAP).

## Goals
- Automate the installation and configuration of Red Hat AAP.
- Ensure the role is modular and follows Ansible best practices.
- Support relevant target environments (Red Hat Enterprise Linux).

## Core Requirements
- Deploy AAP components (Controller, Execution Environments, etc.).
- Integration with existing infrastructure.
- Secure configuration by default.
